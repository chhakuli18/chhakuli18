
# **Installation Guide**
**File:** `Installation_Guide_Sample.md`  

## Software Installation Guide

### Prerequisites
- Operating System: Windows 10 or above
- Minimum RAM: 4 GB
- Disk Space: 2 GB free

### Installation Steps
1. Download the installer from [company site].
2. Run the installer as administrator.
3. Follow the on-screen instructions.
4. Launch the software and activate your license.

### Uninstallation
- Go to Control Panel > Programs > Uninstall.
- Select the software and click **Uninstall**.
