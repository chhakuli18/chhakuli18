# Cloud Storage User Guide

## Introduction
This guide provides step-by-step instructions to use the Cloud Storage Service effectively.

## Features
- Upload and download files
- Share files with others securely
- Organize files into folders

## Getting Started
1. Sign in to your account.
2. Click on **Upload** to add files.
3. Create folders using **New Folder** button.
4. Share files using the **Share** option.

## Troubleshooting
- **Problem:** File not uploading  
  **Solution:** Check internet connection and file size limit.

- **Problem:** Unable to share  
  **Solution:** Ensure recipient has the correct permissions.
